## Programs

### read_file.py
Reads a text file and displays its contents line by line.

**Features:**
- Opens and reads a text file named `sample.txt`
- Prints file content line by line with line numbers
- Handles `FileNotFoundError` gracefully if the file doesn't exist


### write_append.py
Writes user input to a file and appends additional data.

**Features:**
- Takes user input and writes it to a file named `output.txt`
- Appends additional data to the same file
- Reads and displays the final content of the file

# Assignment 2 - Python Programs

## Programs

### 1. EvenOrOdd.py

**Description:** A program that determines whether a given number is even or odd.

**Functionality:**
- Takes an integer input from the user
- Checks whether the number is even or odd using the modulo operator (`%`)
- Displays the result with an appropriate message


### 2. SumOfIntegers1to50.py

**Description:** A program that calculates the sum of all integers from 1 to 50.

**Functionality:**
- Uses a `for` loop to iterate over numbers from 1 to 50
- Accumulates the sum of all integers in this range
- Displays the final sum
